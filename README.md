#
team project Petner !!

### Project 소개
저희 사이트는 펫시팅 전문업체 보다는 가벼운 Casual 한 서비스입니다.

사용자 위치를 기반으로 주변의 펫시터 혹은

반려동물 서비스 현황을 제공하여

주변의 이웃이나 반려동물 보호자도 직접 펫시팅에 참여 할수 있도록 돕는 서비스 입니다.

### 팀원
* 김혜경
* 조다솜
* 최동인
* 김유진
* 홍시원
